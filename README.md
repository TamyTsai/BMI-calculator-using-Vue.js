## 簡介
- [Demo Link]()
- 本專案為一台BMI計算機
- 以HTML、CSS及JavaScript撰寫，為一頁式靜態網頁
- 使用Tailwind CSS設計樣式，並達RWD效果
- 使用Vue.js前端框架製作

## 功能
- 輸入身高、體重（可有小數點）後，網頁將即時出現BMI計算結果
- 未輸入身高或體重時，結果以0顯示
